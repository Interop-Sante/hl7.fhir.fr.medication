Summary

Complex Extension: Specifies descriptive properties of the medicine

  • type: CodeableConcept: A code specifying which characteristic of the medicine is being described (for example, colour, shape, imprint, release characteristic).
  • value: base64Binary, string, Attachment, CodeableConcept, Quantity: The description should be provided as a CodeableConcept, SimpleQuantity or an image. The description can be a string only when these others are not available.
