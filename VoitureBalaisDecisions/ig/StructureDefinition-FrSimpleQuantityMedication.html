Data Type Profile: SimpleQuantity with UCUM or EDQM codes or code not used

Official URL: https://hl7.fr/ig/fhir/medication/StructureDefinition/FrSimpleQuantityMedication Version: 0.1.0
Draft as of 2025-08-21 Computable Name: FrSimpleQuantityMedication

simple quantity datatype requiring a UCUM or EDQM code or no code (only unti)

Usages:

  • Use this DataType Profile: Range with UCUM or EDQM codes if code is used, Ratio with UCUM or EDQM codes if code is used, FR Current Medication MedicationStatement, FR Medication History MedicationStatement... Show 2 more, FR Medication Reconciliation MedicationStatement and FR Medication Request
